Hashing is used in dictionaries where there are key and value pairs 
Also used to implement sets
It provides search, insert and delete all 3 operations in Big O(1) time on average
In hash tables there are always unique values 
No duplicates are allowed 
Overide the previos key if duplicates are implemented
Hashing works best for above 3 operations- Its is an expert data structure in above operations
